###Getting Started###

There are two methods for getting started with this repo.

####Setup with npmt#####
Clone this repo, install dependencies, then start the gulp process with the following:

```
	> git clone https://github.com/gabemartinez/trivia.git
	> cd trivia
	> npm install
	> npm start
```
